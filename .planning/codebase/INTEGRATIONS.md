# Integrations

This codebase uses minimal external API integrations but relies on several UI/styling libraries to create high-fidelity user experiences.

## UI/UX Libraries

- **lucide-react** (v0.546.0): A modern icon library providing vector icons.
- **@headlessui/react** (v2.2.9): Unstyled, fully accessible UI components designed to integrate with Tailwind CSS.
- **@heroicons/react** (v2.2.0): SVG icons for Tailwind CSS projects.
- **aos** (v2.3.4): Animate On Scroll library used for custom enter animations as pages scroll down.

## External APIs & Forms

- **Contact Forms**: Standard HTML forms mapping to custom client actions. No external email/API integration is currently connected (placeholder only).
