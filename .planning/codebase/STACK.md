# Tech Stack

The current project is a modern React web application built with **Next.js 15** and configured for static site generation (SSG) / static HTML export.

## Core Stack

- **Framework**: [Next.js](https://nextjs.org/) (v15.5.5) - React Framework for pre-rendering, routing, and optimization.
- **Library**: [React](https://react.dev/) (v19.1.0) - Core UI library.
- **Language**: [TypeScript](https://www.typescriptlang.org/) (v5.x) - Strongly typed programming language.
- **Styling**: [TailwindCSS](https://tailwindcss.com/) (v4.1.14) - Utility-first CSS framework with native postcss configuration.
- **Routing**: Next.js App Router (located under `src/app/`).

## Deployment / Build Options

- **Static Export**: Currently configured in `next.config.ts` with `output: 'export'`.
- **Target folder**: `out/` - standard directory generated by `next build`.
- **Current Base Path**: `basePath` is set to `/jpl-tech-website` in `next.config.ts`, which is suitable for GitHub Pages but breaks root-domain deployments on cPanel.

## Dev Tools & Linters

- **Eslint**: v9.x configured with standard Next.js lint configurations.
- **PostCSS**: Used with `@tailwindcss/postcss` for tailwind v4 parsing.
